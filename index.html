Showcase

HTML

The backbone of the web. I used it to structure content clearly and semantically, using tags like header, main, section, and article to give each part of the page meaning. Forms, buttons, links, and images were placed with purpose, making sure both humans and machines could understand the layout.

CSS

The soul and personality. It transformed plain structure into an experience. I styled elements with fonts, colors, spacing, and transitions, using Flexbox and Grid to create responsive layouts that worked across screen sizes. Pseudo-classes, keyframe animations, and media queries helped me fine-tune interactivity and design flow.

JavaScript

The brain. I used it to handle user events, update the DOM in real time, and control game logic with loops, conditionals, arrays, and objects. I practiced writing callbacks, explored browser APIs, and applied basic data structures like stacks and queues. Along the way, I sharpened my debugging skills and tackled algorithmic challenges by breaking problems into clear, logical steps.
